### Used Technologies
- Doppler for secrets management
- CloudflareD for DNS and Zero trust
- ElasticSearch & Kibana for logs and traces
- Prometheus & Grafana for metrics
- FluentBit for sending logs to ElasticSearch
- Jaeger for traces
- CNPG for managing Postgres databases
